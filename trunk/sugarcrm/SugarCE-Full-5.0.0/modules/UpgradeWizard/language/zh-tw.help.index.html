Upgrade Wizard

Use this Wizard to upload new patches to upgrade your existing Sugar application. Before you begin this process, ensure that the config.php file for your installation is writable.

  1. Click Next to perform System checks to ensure that you have the necessary file permissions and database privileges to upgrade the system. The status of System Checks changes to In Progress.
  2. Click Next to upload the upgrade files. You can install from the Sugar server or from your local machine.
  3. To upload from the Sugar Exchange:
    1. On the Install from Sugar tab, enter your user name and password, accept the license agreement, and click Login
      If authentication is successful, the system lists available packages on the page.
    2. To install a package, click the corresponding Download button.
      The system downloads the package and displays it below.
    3. Click Next to view the Preflight Check page.
      By default, the Upgrade Wizard Runs SQL option is selected as the database update method. If you ran the SQL queries manually, select Manual SQL Queries from the Database Update Method drop-down list and select the Check when SQL has been manually run box.
    4. Click Next to view the Commit Upgrade page.
      Optionally, you can click Show to see a list of files that were copied and the rebuild results.
    5. Click Next to view the Debrief page.
    6. Click Done.

    To upload from your local machine:
    1. On the Install from local file tab, click Browse to navigate to the location of the upgrade zip file and then click Upload Upgrade; to remove the file, click Delete Package.
    2. Click Next to view the Preflight Check page and follow the process to commit the upgrade and view the Debrief page as described above.
