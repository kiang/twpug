Project Detail Page

Use this page to view project information. Sub-panels for related records such as activities, history, contacts, accounts, opportunities, and quotes display on the page.

  • To edit the project, click Edit, make the necessary revisions, and click Save.
  • To duplicate the information, click Duplicate. The system displays the new record in the list on the Projects Home page. You can then make modifications to the record and save it as a different project.
  • To delete the project, click Delete.
  • View Gantt. Click this button to edit an existing project task or add a new one.
  • Save as Template. Click this button to save the project as a template. Creating a project template enables you to save the associated project tasks for use in other similar projects.
  • Export to MS Project. Click this button to open the project in MS Project or save it as a .mpx file to your local machine.

To add a new record such as a new contact, click Select in the appropriate sub-panel and select the record.

To add project resources, in the Resources sub-panel, click Select User or Select Contact. Users are members of your organizations who are assigned to the project and contacts are other external individuals such as customers or vendors.

To take scheduled holidays and vacations into account, you will need to create a project holiday for the user in the Project Holidays sub-panel. Click Create, enter the holiday date, specify the user from the Resource drop-down list, and click Save.
