Field Type Null Default Links to Comments

trx  text Yes  NULL    Translation Text

Contains the entry to be translated

Field Type Null Default Links to Comments
id  int(11) No      Entry Id 
ref  text No      Text To Translate 
trx  text No      Translated Text 


sugar_mnf

Contains the manifest properties for each versions
(`version_id`) REFER `sugar_lang/sugar_versions`(`id`) ON DELETE CASCADE

Field Type Null Default Links to Comments
version_id  smallint(6) No  0  sugar_versions -> id  Id of the version of the manifest 
lang  varchar(3) No      Indicates if the manifest belongs to a reference package (value 'REF') or to the translated package (value 'TRX') 
name  varchar(40) No      Manifest Property Name 
value  varchar(200) Yes  NULL    Manifest Property Value 


sugar_terms

Contains all the terms of the reference and translated package for each version
(`version_id`) REFER `sugar_lang/sugar_versions`(`id`) ON DELETE CASCADE

Field Type Null Default Links to Comments
id  int(11) No      Sugar Term Id 
version_id  smallint(6) No  0  sugar_versions -> id  Id of the translation version 
file  varchar(100) No      File to which belongs the sugar term 
array_id  varchar(20) No      Name of the array to which belongs the sugar term 
sub_array_id  varchar(50) No      Name of the sub array to which belongs the sugar term (if existing) 
label_id  varchar(50) No      Name of the Label (array key) mapping the term 
lang  varchar(3) No      Indicates if this term is for the reference package (value 'REF') or for the translated package (value 'TRX') 
text  text No      Sugar Term Text
