簡介

Moodle 是一個用來建設基於 Internet 的課程和網站的套裝軟體。這個專案還在不斷繼續開發，其設計目的是提供一個用於教育的 社會建構主義框架。

Moodle 是免費的開放源代碼軟體(遵守 GNU Public License )。簡單說，這意味著 Moodle有版權，但是您仍然有很大的自由度來使用它。您可以拷貝、使用和修改 Moodle，只要您同意：把源代碼提供給他人；不修改和刪除原有的授權協議和版權，並且 其他派生工作也使用同樣的授權協議。如果您有任何問題，請閱讀更詳細的授權協議，並請直接聯繫 版權人。

Moodle 可以在任何支援 PHP 的電腦上運行，並且支援多種類型的資料庫（尤其是 MySQL )。

Moodle 這個詞本是 Modular Object-Oriented Dynamic Learning Environment（模組化面向物件的動態學習環境）的縮寫，程式師和教育理論家經常使用它。它也是一個動詞，表述的是一個這樣的過程：遇事不慌不忙，想起什麼就做什麼，快樂的瞎忙活可也經常會帶來一些領悟和創新。Moodle 的開發就是這樣，學生與教師在線上課程中學習與教課也是這樣。使用 Moodle 的人被稱為Moodler。

快來和我們一起 moodle！
