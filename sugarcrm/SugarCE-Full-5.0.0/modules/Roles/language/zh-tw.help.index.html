Roles

Use this page to search and view existing roles in your organization. A role defines a set of privileges to perform specific tasks. You can assign one or more users to a role to enable them to access the appropriate Sugar modules and perform the tasks.

The Roles Home page displays the following information:

  • A Search sub-panel where you can enter the role name to search for a specific team.
  • A Shortcuts menu that displays the following options:
    • Create Role. Click this option to create a new role. Enter the role name and description and click Save to create the role.
    • List Roles. Click this option to display a drop-down list of users. Select a user from the list to view the role and associated access permissions granted to the user in Sugar.
    • List Roles by User. Click this option to view access permissions for a specific user, and select the user from the adjacent drop-down list.
  • A list of existing roles along with their description.
    • To view the details of a role, click the name in the Roles list.
